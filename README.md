# Health Up

**Health Up** is a comprehensive health and wellness platform designed to help users track, manage, and improve their lifestyle. Whether you're monitoring your fitness goals, tracking nutrition, or managing appointments, Health Up provides an intuitive interface and a suite of powerful features for individuals and healthcare professionals.

## Features

- **User Dashboard**: Personalized health summaries and actionable insights.
- **Activity Tracking**: Log workouts, steps, and calories burned.
- **Nutrition Management**: Record meals, get dietary suggestions, and analyze nutrition intake.
- **Appointment Scheduling**: Easily book, view, and manage appointments with healthcare providers.
- **Health Records**: Securely store and access medical reports and vaccination history.
- **Notifications**: Set up reminders for medication, appointments, or daily activities.

## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/saran125/Health_up.git
    cd Health_up
    ```

2. **Install dependencies**
    ```bash
    # Frontend
    cd frontend
    npm install

    # Backend
    cd ../backend
    pip install -r requirements.txt
    ```

3. **Configure environment variables**
    - Copy `.env.example` to `.env` and fill in your local configuration.

4. **Run the app**
    ```bash
    # Backend
    python manage.py runserver

    # Frontend
    npm start
    ```

5. **Open your browser**
    ```
    http://localhost:3000
    ```
